## 📊 결과 해석 가이드

### 🎯 신뢰도 등급
- **90%+**: 🟢 매우 높음 - 강력한 거래 신호
- **80-90%**: 🟢 높음 - 거래 신호 활성
- **60-80%**: 🟡 보통 - 관찰 권장
- **60% 미만**: 🔴 낮음 - 대기 권장

### 📈 예측 신호 종류
- **"강한 매수 신호"**: 상승 예상, 높은 확률
- **"강한 돌파 신호"**: 큰 움직임 예상
- **"조정 예상"**: 하락 조정 예상
- **"중립"**: 큰 변화 없음

### ⚡ 패턴 조합
- **Momentum + Volume 둘 다 강함**: 최고 신뢰도
- **Momentum만 강함**: 추세 신호
- **Volume만 강함**: 돌파 신호

## 🔧 고급 사용법

### 프로그래밍 방식 사용
```python
from btc_price_predictor import BTCPricePredictor
import asyncio
from datetime import datetime, timedelta

async def my_prediction():
    predictor = BTCPricePredictor()
    
    # 내일 오후 3시 예측
    target_time = datetime.now().replace(hour=15, minute=0) + timedelta(days=1)
    result = await predictor.predict_future_price(target_time)
    
    print(f"예측 가격: ${result['predicted_price']}")
    print(f"신뢰도: {result['confidence']}%")

asyncio.run(my_prediction())
```
