# 🔧 Git 자동화 로그 시스템 사용법 (초보자용)

> "코딩하다가 뭔가 잘못되면 어떻게 되돌리지?" 걱정 끝! 🎉
> 
> 이 시스템은 여러분의 모든 작업을 자동으로 저장하고, 언제든 되돌릴 수 있게 해줍니다.

---

## 🌱 처음 사용자를 위한 3분 시작 가이드

### 1단계: 초기 설정 (처음 한 번만)
```bash
# 터미널에서 다음 명령어 실행
./setup_git_system.sh
```

**이름과 이메일을 물어보면 입력하세요** (Git에서 사용하는 정보예요)

### 2단계: 작업 시작하기
```bash
# 새로운 작업을 시작할 때
./start_work.sh
```

**"작업 내용을 입력해주세요"라고 나오면 예를 들어:**
- "로그인 기능 만들기"
- "버튼 색깔 바꾸기"  
- "데이터베이스 연결"

### 3단계: 작업 완료하기
```bash
# 작업을 완료했을 때
./complete_work.sh
```

**"완료한 내용을 입력해주세요"라고 나오면 예를 들어:**
- "로그인 기능 완성"
- "버튼 색깔 파란색으로 변경 완료"
- "MySQL 데이터베이스 연결 성공"

## 🚨 문제가 생겼을 때 (되돌리기)

### 최근 작업으로 되돌리기 (가장 많이 사용)
```bash
./rollback_work.sh
# 1번 선택 → 최근 작업으로 되돌리기
```

### 특정 작업으로 되돌리기
```bash
./rollback_work.sh  
# 2번 선택 → 작업 번호 입력 (예: 3)
```

### 오늘 시작 시점으로 되돌리기
```bash
./rollback_work.sh
# 3번 선택 → 오늘 첫 번째 작업으로
```

## ❓ 자주 묻는 질문 (FAQ)

### Q: "git이 뭔지 몰라도 사용할 수 있나요?"
**A: 네! Git을 전혀 몰라도 괜찮아요.** 
- 스크립트가 모든 복잡한 것들을 대신 해줍니다
- 그냥 `./start_work.sh`와 `./complete_work.sh`만 기억하세요

### Q: "실수로 파일을 삭제했어요!"
**A: 되돌리기를 사용하세요:**
```bash
./rollback_work.sh
# 1번 선택하면 최근 작업으로 되돌아갑니다
```

### Q: "작업 번호가 뭔가요?"
**A: 작업마다 자동으로 매겨지는 번호예요:**
- 첫 번째 작업: #1
- 두 번째 작업: #2  
- 세 번째 작업: #3
- `./check_status.sh`로 확인할 수 있어요

### Q: "에러 메시지가 나와요"
**A: 가장 흔한 해결방법들:**

1. **"git command not found"**
   ```bash
   # macOS의 경우
   xcode-select --install
   ```

2. **"permission denied"**  
   ```bash
   chmod +x *.sh
   ```

3. **"working tree is dirty"**
   ```bash
   ./rollback_work.sh  # 최근 상태로 되돌리기
   ```

### Q: "GitHub에 올리고 싶어요"
**A: 수동으로 해야 해요 (안전을 위해):**
```bash
# 1. GitHub에서 레포지토리 만들기
# 2. 다음 명령어 실행
git remote add origin https://github.com/본인아이디/프로젝트명.git
git push -u origin main
```

## 🎯 사용 패턴 예시

### 하루 일과 예시:
```bash
# 아침에 출근
./setup_git_system.sh        # (처음 한 번만)

# 첫 번째 작업 시작  
./start_work.sh              # "메뉴 버튼 만들기"

# (코딩 작업...)

# 첫 번째 작업 완료
./complete_work.sh           # "메뉴 버튼 완성"

# 두 번째 작업 시작
./start_work.sh              # "로그인 폼 추가"

# (코딩 작업 중 문제 발생!)

# 되돌리기
./rollback_work.sh           # 1번 선택

# 다시 시작
./start_work.sh              # "로그인 폼 다시 만들기"

# 퇴근 전
./daily_backup.sh            # 하루 백업
```
